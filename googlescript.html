Scripts

Purpose:

Automate processes like collecting, organizing, and sending out data.

Execution:

I created many different scripts to automate tedious processes that were needed to set up documents at the tutoring center I work for. For example, to send out survey data to our tutors, we need to pull the responses, sort through them to pull out each individual tutor's responses, create documents for each tutor (since tutors can't see each other's feedback), and send those documents out. I wrote a script that creates tabs with a custom formula on each tab for each tutor. This formula parses all the data in another tab and queries for a certain tutor's name. Then, my script iterates through the tabs and creates a pdf with an appropriate name for each tab, sending out an email with the correct pdf to the correct tutor. A process that would have taken hours is now reduced to a few minutes.
